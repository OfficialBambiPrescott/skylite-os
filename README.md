# Skylite OS & EdgeFront – Open-Source Beta

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Build Status](https://github.com/yourusername/skylite-os/workflows/CI/badge.svg)](https://github.com/yourusername/skylite-os/actions)

## Overview
Skylite is a conceptual, privacy-first OS ecosystem inspired by Microsoft Windows, with #EdgeFront scripting for Zero Trust apps. This beta release includes prototypes for the EdgeFront VM, sample scripts (e.g., Secure Notes, Health Check), and browser skeletons.

- **Philosophy**: Epicurean verification, Newtonian laws, Sacred Geometry harmony.
- **Math Core**: Newton-Raphson crypto, Euclidean permissions, Vesica Piscis UI.
- **Status**: Beta – proof-of-concept. Contributions welcome!

## Installation
1. Clone: `git clone https://github.com/yourusername/skylite-os.git`
2. Build EdgeFront VM (C++): Use Visual Studio – see src/edgefront/vm.cpp
3. Run scripts: `./edgefront secure_notes.ef`

## Usage
- Run Secure Notes: `edgefront src/edgefront/secure_notes.ef` – encrypted notepad demo.
- Explore docs/architecture.md for full specs.
